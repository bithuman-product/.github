## Get started

The fastest path is **[docs.bithuman.ai/start](https://docs.bithuman.ai/start)** — see an agent talk with no setup, then pick how you run it.

| You want to… | Go to | What you get |
|---|---|---|
| **Try it in ~2 minutes** | [Quickstart](https://docs.bithuman.ai/start) | A live, talking avatar — embed one instantly (no key), then run your own. |
| **Run it on your own hardware** | [SDK](https://docs.bithuman.ai/sdk) | Python, Swift (Apple), the browser, or a single CLI command — on-device. |
| **Python SDK** | [`pip install bithuman`](https://pypi.org/project/bithuman/) → [docs](https://docs.bithuman.ai/sdk/python) | The GA on-device runtime. Any CPU — laptop, server, Raspberry Pi. |
| **Mac CLI (voice + video in Terminal)** | [`brew install …`](https://github.com/bithuman-product/homebrew-bithuman) → [docs](https://docs.bithuman.ai/sdk/cli/install) | 100% on-device voice + video chat from the command line, no code. |
| **Swift (iOS / iPadOS / macOS)** | [Swift SDK](https://docs.bithuman.ai/sdk/swift) | On-device, Apple Silicon — `import bitHumanKit`. |
| **Browser / web** | [JS/TS](https://docs.bithuman.ai/sdk/javascript) · [WebAssembly](https://docs.bithuman.ai/sdk/wasm) | Run agents in the browser, or connect a web client to a hosted avatar. |
| **Call the cloud REST API** | [API](https://docs.bithuman.ai/api/quickstart) · [Reference](https://docs.bithuman.ai/api/reference) | No local runtime — generate, host, and drive avatars over HTTPS from any language. |

## Two avatar engines

|  | **Essence** — start here | **Expression** |
|---|---|---|
| **What it is** | A ready-made avatar in a portable `.imx` file | Turns any portrait into a talking agent at runtime |
| **Runs on** | Any modern CPU · Apple Silicon · the browser (WASM) | NVIDIA GPU or recent Apple Silicon |
| **Best for** | Voice agents, kiosks, 24/7 displays, getting started | Apps where users bring their own face |

Compare them and see pricing → **[docs.bithuman.ai/concepts/models](https://docs.bithuman.ai/concepts/models)** · **[pricing](https://docs.bithuman.ai/guides/pricing)**.

## Runs where you need it

`iPhone` · `iPad` · `Mac` · `Linux` · `Raspberry Pi` · `the browser` · `NVIDIA GPU` · `our cloud` — the same `.imx` avatar plays back identically everywhere.
